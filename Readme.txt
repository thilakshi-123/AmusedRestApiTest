How to Run The tests.

Versions 
Visual Studio - Microsoft Visual Studio Community 2022
Dotnet Version - 8.0.302
c# version - 12.0

Step 1
Open the project and add the NuGet packages RestSharp,Xunit and FluentAssertions

Step 2
Build the project - Build > Build Solution

Step 3
Run the test
Test > Test Explorer > Run

For code Verifications please navigate to AmuTestOne.cs

Obtained Results
No of Executed tests - 08
No of Passed tests - 05
No of failed tests - 03
